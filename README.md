This repository contains a Treemap visualization created using RAWGraphs to analyze market share distribution across product categories and subcategories.

## Description

The Treemap visualizes hierarchical market data in an intuitive, publication-ready format suitable for executive presentations and strategic planning. Each rectangle represents a subcategory, sized according to its market value and grouped by category. A professional color scheme and clear labels ensure easy interpretation by non-technical stakeholders.
